# Simon Says Game

🚀 Live Demo: https://simon-says-game-l0dq.onrender.com

## Description
Simon Says is a memory-based game built using JavaScript, HTML, and CSS. The game generates a sequence of colors that the player must repeat correctly. With each level, the sequence becomes longer, increasing the difficulty and testing the player's memory.

## Features
- Random color sequence generation
- Increasing difficulty levels
- Interactive UI with visual feedback
- Score tracking system

## Technologies Used
- JavaScript
- HTML
- CSS
